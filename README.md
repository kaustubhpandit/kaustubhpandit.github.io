# Kaustubh Pandit — Bioimaging Research Portfolio

Personal portfolio website for academic research in OCT, Ultrasound, ULM, and Photoacoustic Imaging.

## 🚀 Deploy to GitHub Pages

1. Create a new GitHub repository named `[your-username].github.io` (for root domain) OR any name (for `[username].github.io/[repo-name]`)
2. Push all files in this folder to the `main` branch
3. Go to **Settings → Pages** → set source to **Deploy from branch → main → / (root)**
4. Your site will be live at `https://[your-username].github.io`

## 📁 File Structure

```
├── index.html          # Home / Hero page
├── research.html       # Research & Projects
├── publications.html   # Publications list
├── news.html           # News & Updates
├── contact.html        # Contact form
├── assets/
│   ├── css/styles.css  # All styles
│   └── js/main.js      # Canvas animations + interactions
├── .nojekyll           # Prevents Jekyll processing
└── _config.yml
```

## 🎨 Theme

Dark scientific theme: deep navy (#050d1a) + electric cyan (#00d4ff) + violet (#7b2fff)

Canvas animations simulate: OCT B-scan, Ultrasound B-mode fan, ULM microbubble tracking, Photoacoustic wave propagation.
